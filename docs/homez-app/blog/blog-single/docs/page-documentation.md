# Blog Single - Artículo Individual

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | blog-single |
| **Categoría** | blog |
| **URL** | /(blog)/blogs/[id] |

## Descripción

Página de artículo individual del blog con contenido completo, imagen destacada, autor y comentarios.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(blog)/blogs/[id]/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
