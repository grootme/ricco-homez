# Blog List V1 - Listado de Blog (Variante 1)

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | blog-list-v1 |
| **Categoría** | blog |
| **URL** | /(blog)/blog-list-v1 |

## Descripción

Primera variante de listado de artículos del blog con diseño de tarjetas grandes y sidebar.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(blog)/blog-list-v1/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
