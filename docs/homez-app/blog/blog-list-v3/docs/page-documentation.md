# Blog List V3 - Listado de Blog (Variante 3)

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | blog-list-v3 |
| **Categoría** | blog |
| **URL** | /(blog)/blog-list-v3 |

## Descripción

Tercera variante de listado del blog con diseño minimalista y lista de artículos.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(blog)/blog-list-v3/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
