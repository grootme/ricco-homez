# Homez - Real Estate React NextJS Template

A premium, modern designed Real Estate React NextJS Template that provides a comprehensive solution for running a realtor company, real estate agency, broker platform, agents, and related businesses.

**Live Preview:** [https://homez-app-preview.vercel.app/](https://homez-app-preview.vercel.app/)  
**Demo App:** [https://homez-appdir.vercel.app/](https://homez-appdir.vercel.app/)

## Project Overview

**Name:** Homez  
**Type:** Premium Real Estate React NextJS Template  
**Purpose:** A comprehensive real estate template for building professional property listing and management websites.

Homez is a premium Real Estate React NextJS Template that lets you run a realtor company, a real estate agency, a broker, agents and related businesses. It features modern design, extensive page layouts, and comprehensive property management functionality.

## Features

### Core Features

1. **Mortgage Calculator** - Built-in widget for calculating mortgage payments for properties
2. **Compare Property** - Compare multiple properties against each other by features and parameters
3. **Saved Properties** - Users can save favorite properties to their account
4. **Page Statistics** - Property owners and agencies can view audience numbers on their listings
5. **Agencies & Agents** - Register as agency, add agents and manage listings
6. **Mobile Friendly** - Retina ready and fully responsive design
7. **360 Virtual Tour** - Immersive virtual property tours
8. **Floor Plans** - Upload and display property floor plans
9. **Nearby Listings & Walk Score** - Overview of nearby amenities

### User Dashboard Features

- **Submit Properties** - Add new property listings
- **Manage Properties** - Edit, update, delete listings
- **User Profiles** - Manage account information
- **Manage Agents/Agencies** - Agent and agency management
- **Payment Integration** - Payment processing functionality
- **Membership System** - Subscription and membership tiers
- **Saved Searches** - Save search criteria for future use
- **Private Message** - Direct messaging system

### Advanced Search Functionality

- Filter by location
- Property type selection
- Price range slider
- Bedroom/bathroom count
- Square footage range
- Amenities filtering
- Save search functionality

### Setup Steps

```bash

# Navigate to project directory
cd homez

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
