# Agents - Listado de Agentes

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | agents |
| **Categoría** | agents |
| **URL** | /(agents)/agents |

## Descripción

Página de listado de agentes inmobiliarios con búsqueda, filtros por especialidad y tarjetas de perfil con estadísticas.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(agents)/agents/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
