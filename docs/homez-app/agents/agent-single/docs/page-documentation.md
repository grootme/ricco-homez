# Agent Single - Perfil de Agente

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | agent-single |
| **Categoría** | agents |
| **URL** | /(agents)/agent-single/[id] |

## Descripción

Página de perfil individual de agente con información biográfica, listado de propiedades, estadísticas de ventas y formulario de contacto.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(agents)/agent-single/[id]/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
