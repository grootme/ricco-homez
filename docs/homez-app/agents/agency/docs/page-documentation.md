# Agency - Listado de Agencias

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | agency |
| **Categoría** | agents |
| **URL** | /(agents)/agency |

## Descripción

Página de listado de agencias inmobiliarias con búsqueda y tarjetas de información de cada agencia.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(agents)/agency/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
