# Property Single Vv9 - Detalle de Propiedad #v9

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | single-v9 |

## Descripción

Página de detalle de propiedad variante v9 con galería de imágenes, información de precio, características, mapa de ubicación, agente contact y propiedades similares. Incluye secciones de detalles, amenities, cercanías y formulario de contacto.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(property)/single-v9/[id]/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
