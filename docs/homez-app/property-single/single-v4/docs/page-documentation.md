# Property Single Vv4 - Detalle de Propiedad #v4

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | single-v4 |
| **Categoría** | property-single |
| **URL** | /(property)/single-v4/[id] |

## Descripción

Página de detalle de propiedad variante v4 con galería de imágenes, información de precio, características, mapa de ubicación, agente contact y propiedades similares. Incluye secciones de detalles, amenities, cercanías y formulario de contacto.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(property)/single-v4/[id]/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
