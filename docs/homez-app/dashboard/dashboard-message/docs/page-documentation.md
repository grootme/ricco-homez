# Dashboard Messages - Mensajes

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | dashboard-message |
| **Categoría** | dashboard |
| **URL** | /(dashboard)/dashboard-message |

## Descripción

Centro de mensajes del usuario con lista de conversaciones, chat en tiempo real y notificaciones.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(dashboard)/dashboard-message/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
