# Dashboard My Properties - Mis Propiedades

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | dashboard-my-properties |
| **Categoría** | dashboard |
| **URL** | /(dashboard)/dashboard-my-properties |

## Descripción

Listado de propiedades publicadas por el usuario con opciones de editar, eliminar y estadísticas de vistas.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(dashboard)/dashboard-my-properties/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
