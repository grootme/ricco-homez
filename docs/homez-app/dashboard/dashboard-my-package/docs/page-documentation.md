# Dashboard My Package - Mi Paquete

## Información General

| Campo | Valor |
|-------|-------|
| **Página** | dashboard-my-package |
| **Categoría** | dashboard |
| **URL** | /(dashboard)/dashboard-my-package |

## Descripción

Información del paquete de suscripción del usuario con detalles de planes y renovación.

## Recursos

- **Video**: `videos/` - Grabación WebM
- **Imágenes**: `images/` - Capturas de pantalla
- **Datos**: `docs/` - JSON y documentación técnica

## Ruta de Código

`src/app/[locale]//(dashboard)/dashboard-my-package/page.tsx`

## Reconstrucción

1. Copiar estructura de `page.tsx`
2. Importar componentes de `@/components/`
3. Configurar datos mock o API
4. Verificar responsividad móvil/desktop
5. Comparar con video e imágenes de referencia
